Meant for trianing models to be used in sweepy but ig it doubles as a deck builder.    
Used https://github.com/hzyhhzy/UmaAi as Reference and sweepy's training logic to simulate careers.      
Things like inspo/shrine/race fatigue not implemented, career is very barebones but eh gets the job done.    
Also rank cards based on existing deck. (Do not set sample size to be under 2000, it will become inaccurate) its comically slow but eh (its not stuck it actually takes that long).
